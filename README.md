## 🚀 Proyectos Destacados

### 🌡️ Calculadora de Enfriamiento
Un script diseñado para modelar y calcular el tiempo que tarda un cuerpo (como una comida) en enfriarse hasta una temperatura deseada.
*   **Conceptos aplicados:** Modelado matemático (Ley de enfriamiento de Newton), resolución de ecuaciones y variables dinámicas.
*   **Visualización:** El programa genera automáticamente una gráfica que muestra la curva de enfriamiento exponencial a lo largo del tiempo, permitiendo analizar visualmente el proceso termodinámico.

## ⚙️ Cómo ejecutar
Para probar los scripts, simplemente abre el archivo `.m` correspondiente en tu entorno de MATLAB o MATLAB Online y haz clic en "Run" (o presiona F5).
